# What's Working?

I can't possibly list everything but some points of note include:

- The compiler, which produces friendly error messages.
- States and state expressions, which update attributes.
- The layout system.
- Text layouts (Although laggy, see [issue](https://github.com/IrisChase/IVD/issues/2)).
- Dynamic model binding.
- Animations of arbitrary scalar values.
- The expression solver (Not thoroughly tested).
